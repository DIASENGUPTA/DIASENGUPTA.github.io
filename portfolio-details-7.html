Project information

  • Name: Path Planning and Robot Control for Autonomous Driving
  • Category: Robotics
  • Project date: Feb 2023 - Mar 2023
  • Project URL: https://github.com/DIASENGUPTA/Robotics-Projects

Project Detail

  • Designed a fully functional autonomous driving vehicle in CARLA with homography transformation, lane and waypoint prediction and target speed prediction based on path smoothness.
  • Optimized lateral control using the Stanley controller and fine-tuned longitudinal control through a PID controller.
